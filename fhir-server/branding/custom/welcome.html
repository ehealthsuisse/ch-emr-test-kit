Welcome to the eHealth Suisse — CH EMR Test Kit — a disposable HAPI FHIR server for experimenting with Swiss FHIR Implementation Guides (e.g. CH EMR).

This server is operated for testing purposes by/for eHealth Suisse, the competence and coordination body of the Confederation and the cantons for digital health. It loads Implementation Guides on startup and exposes this generic tester.

It is built on the open-source HAPI FHIR stack (Apache 2.0). Data on this server is for testing only and may be reset at any time.
